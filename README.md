## Overview

Predictive credit risk assessment system using LSTM neural networks and machine learning models. Built during role at **Mizuho Bank** to enable real-time credit risk evaluations within digital platforms.

## Key Features

- **LSTM Models** for feature extraction and credit risk prediction
- **RESTful APIs** using Flask for real-time credit risk assessments
- **ETL Pipelines** using Apache Airflow to process and cleanse financial datasets
- **Generative AI** techniques to create synthetic datasets addressing data privacy
- **CI/CD Pipelines** using Jenkins for automated testing and deployment
- **Azure Kubernetes Service (AKS)** for efficient scaling
- **Model Monitoring** with Prometheus and Grafana for real-time performance tracking
- **Streamlit UI** for interactive AI-driven credit risk insights
- **GDPR & Regulatory Compliance** built-in

## Architecture

```
Data Ingestion (Airflow ETL)
    ↓
Feature Engineering + LSTM Training (Azure ML)
    ↓
Model Deployment (AKS via Jenkins CI/CD)
    ↓
Flask REST API → Streamlit Dashboard
    ↓
Prometheus + Grafana Monitoring
```

## Results

- Real-time credit risk assessments deployed in digital platforms
